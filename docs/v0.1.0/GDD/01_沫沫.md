# 沫沫（momo）角色细节设计

## 1. 角色定位

| 属性 | 内容 |
| --- | --- |
| 中文名 | 沫沫 |
| 内部代号 | momo |
| 生成群系 | 繁华森林（Flower Forest） |
| 维度 | 主世界 |
| 核心主题 | 花草、蜜蜂、温室、自然陪伴 |
| 性格特质 | 温柔细腻、热爱自然、表面柔弱但内心坚韧、有耐心 |
| 说话风格 | 轻柔温和，喜欢用植物的比喻，偶尔自言自语关于花草的事 |
| 声音特点 | 声线柔软清澈，语速偏慢，尾音轻轻下落，像晨雾里沾着露水的花瓣；情绪起伏不大，但在谈到花园、蜂群与“家”时会自然变得更温暖 |
| 核心愿望 | 在纷乱世界中建起一座不受风雨影响的温室花园，让植物在任何天气里都能安稳生长 |
| 内心矛盾 | 喜欢照料生命但害怕失去它们，因此最初对“把植物种在外面”心怀隐忧 |

## 2. 外观设计方向

| 维度 | 方向描述 |
| --- | --- |
| 整体印象 | 柔和自然系，色调以银色、米白、鹅黄为主 |
| 发型 | 及肩的银白色长发，常戴着几朵小花编制的花环 |
| 服装 | 藏蓝色的格裙，外搭米色外衣，外衣口袋常装着种子 |
| 配饰 | 手腕上有藤蔓编织的手环，有时会戴一顶小草帽 |
| 随身物品 | 手持小铲或花洒，背包里总有多余的种子和花肥 |
| 身高体态 | 中等偏娇小，举止轻盈，走路时裙摆轻轻摆动 |

## 3. 庇护所设计

| 维度 | 描述 |
| --- | --- |
| 名称 | 花影小筑 |
| 结构 | 小型木制小屋，外墙爬满藤蔓与花朵，屋顶有蜂箱 |
| 周边 | 小屋外有围栏圈出的小花圃，种着各色花草，角落有堆肥桶 |
| 内部 | 简洁温馨，窗台摆满盆栽，墙上有干花装饰，角落有育种台 |
| 成长变化 | 花圃范围逐步扩大 → 温室框架出现 → 温室建成 → 内部植物种类丰富 → 增设休憩长椅与观赏区 |
| 氛围 | 宁静、生机勃勃、有蜜蜂的嗡嗡声和花香 |

## 4. 日常行为模式

| 时段 | 行为 |
| --- | --- |
| 清晨 | 在花圃中浇水、观察新芽 |
| 上午 | 采集花朵、照料蜂箱、整理苗床 |
| 下午 | 在庇护所周边散步、收集种子、观察野生花草 |
| 傍晚 | 回到庇护所整理一天的收获、制作干花 |
| 夜晚 | 在屋内休息，偶尔在窗边看月光下的花园 |
| 雨天 | 会在温室内（建成后）或庇护所檐下避雨，担忧外面的花草 |

## 5. 性格深度

沫沫对自然的热爱源于一种朴素的信念：每一种生命都值得被温柔对待。她不是那种大声宣扬自己理念的人，而是用行动默默照料每一株植物、每一只蜜蜂。

她的内心隐忧来自于过去的经历——她曾精心照料的花圃在一场暴风雨中被毁，从此她对“露天种植”心怀不安，渴望建一座温室来保护植物。这也是她核心愿望的由来。

在与玩家的关系中，沫沫最初只是感激有人愿意帮她整理花圃。随着委托推进，她会逐渐把玩家视为“能让花园变得更安全的人”，并最终希望玩家成为“一起照料这座花园的人”。她的亲密表达不是热烈的告白，而是“我想把花园的钥匙交给你”。

## 6. 固定委托详设

### 委托 1：花影初识

| 维度 | 描述 |
| --- | --- |
| 阶段 | 熟悉（好感度 100~200） |
| 触发条件 | 玩家好感度达到 100，且当前委托槽为空 |
| 委托目标 | ① 为沫沫收集 3 种不同的花朵（任意品种）② 为她收集 3 棵树苗（任意品种）③ 在庇护所周边清理出一块 5×5 的平整空地，并用橡木栅栏围起来 |
| 任务指引 | 任务卡文字：“你能帮我收集一些花和树苗吗？我想在屋子旁边辟一小块地方……” |
| 完成反馈 | 沫沫在空地周围插上小栅栏，撒下第一批种子。她会站在空地边轻声说：“如果有一个不会受风雨影响的地方，它们会长得更好吧。” |
| 环境变化 | 庇护所边出现一块被栅栏围起的小花圃，里面有刚种下的幼苗 |
| 主要奖励 | 好感度 +50；解锁「花朵采集」「蜂蜜补给」随机委托模板 |
| 协助可能 | 其他玩家可以帮助收集材料，但只有接取者获得奖励与推进 |

## 9. 祝福、跟随与终章方向

| 项目 | 内容 |
