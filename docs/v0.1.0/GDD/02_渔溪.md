# 渔溪（yuxi）角色细节设计

## 1. 角色定位

| 属性 | 内容 |
| --- | --- |
| 中文名 | 渔溪 |
| 内部代号 | yuxi |
| 生成群系 | 沙滩（Beach） |
| 维度 | 主世界 |
| 核心主题 | 海洋、航行、钓鱼、静谧陪伴 |
| 性格特质 | 安静内敛、善于倾听、内心藏着对远方的浪漫向往 |
| 说话风格 | 简洁平静，偶尔说出富有画面感的短句，喜欢用“风”“浪”“岸”等意象 |
| 声音特点 | 声线偏清冷，音量不高，咬字轻却很稳，像海风贴着耳边掠过；说到远航与归处时会多出一点柔软的呼吸感，让人觉得她把重要的话都藏在浪声后面 |
| 核心愿望 | 与信任的人一同走遍天涯海角，看尽所有未曾见过的海岸线 |
| 内心矛盾 | 渴望远航但害怕独自面对未知海域的危险，需要一个可以同行的伙伴 |

## 2. 外观设计方向

| 维度 | 方向描述 |
| --- | --- |
| 整体印象 | 清爽海洋系，色调以海蓝、白色、沙色为主 |
| 发型 | 深蓝色长直发，扎成低马尾，额前有几缕碎发被海风吹散 |
| 服装 | 白色短袖上衣搭配深蓝色长裙，外罩轻薄的长开衫，衣摆随海风飘动 |
| 配饰 | 脖子上挂着一枚鹦鹉螺壳吊坠，是她最珍视的物品 |
| 随身物品 | 手持钓竿，腰间挂着一个小布袋 |
| 身高体态 | 中等身高，体态修长，站姿安静沉稳，像海边的一块礁石 |

## 3. 庇护所设计

| 维度 | 描述 |
| --- | --- |
| 名称 | 潮声小屋 |
| 结构 | 海边木制小屋，屋顶铺着晒干的棕榈叶，门前有简易码头 |
| 周边 | 屋旁停着一条小船，晾晒架上挂着渔网和鱼干，沙滩上有贝壳收集区 |
| 内部 | 简朴但有序，墙上挂着海图，窗台摆着漂流瓶和贝壳，角落堆着渔具 |
| 成长变化 | 小船修整 → 码头扩建 → 渔具增多 → 海图逐渐完整 → 增设灯塔灯饰 → 屋内增加更多旅途收藏 |
| 氛围 | 安静、潮汐声、海风轻拂、偶尔有海鸥鸣叫 |

## 4. 日常行为模式

| 时段 | 行为 |
| --- | --- |
| 清晨 | 在岸边散步，检查小船和渔网 |
| 上午 | 在码头或岸边垂钓，偶尔下水采集海产品 |
| 下午 | 整理渔获、修补渔网、在沙滩上绘制简易海图 |
| 傍晚 | 坐在码头边看日落，收集漂流物 |
| 夜晚 | 在屋内整理一天的渔获，借着灯光看海图 |
| 雨天 | 在庇护所檐下安静地看雨落在海面，修补渔具 |

## 5. 性格深度

渔溪是一个习惯安静的人。她不是在回避交流，而是更善于用陪伴而非言语来表达心意。当她信任一个人时，她会邀请对方一起坐在码头边，不说什么话，只是一起钓鱼，一起看海。

她对远方的向往来自一枚鹦鹉螺壳——那是她小时候在海边捡到的，壳里的回声让她想象着千万里之外的海域是什么样子。她想亲眼看一看，但又对独自远航心怀犹豫。玩家是她第一个觉得“可以一起出海”的人。

渔溪的亲密表达不是热情的表白，而是“下一片海，我们一起去”。当她最终决定在玩家家园中留下归航灯时，她的意思是：无论航行多远，这里都是回来的方向。

## 6. 固定委托详设

### 委托 1：潮声边的问候

| 维度 | 描述 |
| --- | --- |
| 阶段 | 熟悉（好感度 100~200） |
| 触发条件 | 玩家好感度达到 100，且当前委托槽为空 |
| 委托目标 | ① 为渔溪收集 3 根钓鱼竿或制作材料 ② 收集 16 个可用于小舟修补的木板 ③ 为她准备 8 个食物（任意可食用物品） |
| 任务指引 | 任务卡文字：“我的钓竿快不行了……如果你能帮我找些材料，或许我能修好它，再准备一些出行的东西。” |
| 完成反馈 | 渔溪接过材料，安静地在码头边修整渔具。她抬头看了一眼前方的海面：“你想在这里坐一会儿吗？潮声很好听。” |
| 环境变化 | 码头边的渔具变得更加完好，小船旁多了一些补给箱 |
| 主要奖励 | 好感度 +50；解锁「常见渔获」「水边采集」随机委托模板 |
| 协助可能 | 其他玩家可以帮助收集材料 |

### 委托 2：午后河岸

| 维度 | 描述 |
| --- | --- |
| 阶段 | 信赖（好感度 200~350） |
| 触发条件 | 完成委托 1，好感度达到 200 |
| 委托目标 | ① 与渔溪一同前往指定水域 ② 在水域旁陪伴至少半个游戏日（期间可自由钓鱼）③ 钓到至少 5 条任意鱼类 |
| 任务指引 | 任务卡文字：“有一条河的出海口，那里的鱼群最近很活跃。你想一起去看看吗？不着急，我们可以待久一点。” |
| 完成反馈 | 渔溪坐在玩家旁边安静地钓鱼。收竿时她会说：“你知道吗，每片海的声音都不一样。我希望有一天能听遍所有的海。” |
| 环境变化 | 渔溪的庇护所中会多出一条新钓上来的鱼作为“纪念” |
| 主要奖励 | 好感度 +50；奖励各种鱼类资源；解锁「小舟整备」随机委托模板 |
| 协助可能 | 其他玩家可陪伴钓鱼，但不能替代完成 |

| 环境变化 | 庇护所附近出现更完整的航海用品摆设，码头更加规整 |

### 委托 4：海风记事

| 维度 | 描述 |
| --- | --- |
| 阶段 | 信赖（好感度 500） |
| 触发条件 | 完成委托 3，好感度达到 500 |

| 完成反馈 | 渔溪小心翼翼地将纪念品摆放在庇护所的陈列架上。她拿起一枚珊瑚看了很久：“每件东西后面都有一段路。这些路，以后我也想亲自走一遍。” |
| 环境变化 | 庇护所内新增地图、漂流物与海边收藏展示架 |
| 主要奖励 | 好感度 +50；奖励海洋探索类资源（海晶碎片、珊瑚块、鹦鹉螺壳等） |
| 协助可能 | 其他玩家可以帮助走访海岸和收集纪念品 |

### 委托 5：更远的岸

| 维度 | 描述 |

| 环境变化 | 渔溪的海图上多出一条新的航线标记，庇护所内增加该海域的地图标记 |

### 委托 8：与你看尽远方

| 触发条件 | 完成委托 7，好感度达到 700；玩家需要主动与渔溪交互选择“确认关系” |

| 协助可能 | 其他玩家无法参与此委托 |

### 委托 10：留一盏归航灯

## 9. 祝福、跟随与终章方向

| 项目 | 内容 |
