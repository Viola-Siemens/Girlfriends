# 梅疏（meishu）角色细节设计

## 1. 角色定位

| 属性 | 内容 |
| --- | --- |
| 中文名 | 梅疏 |
| 内部代号 | meishu |
| 生成群系 | 尖峭山峰（Jagged Peaks） |
| 维度 | 主世界 |
| 核心主题 | 采矿、矿道、材料、合金愿望 |
| 性格特质 | 沉默寡言、专注执着、不善表达但行动力强、对矿石有近乎本能的直觉 |
| 说话风格 | 短句为主，惜字如金，但谈及矿石和材料时会多说几句，用词精准 |
| 声音特点 | 声线低而干净，语气克制，句子短促有力，像敲击矿石时清晰而稳定的回响；平时很少起伏，但在试验成功或把成果交给玩家时，会显出少见的温度和停顿 |
| 核心愿望 | 研制出足够坚硬的合金材料，证明自己对大地与矿石的理解 |
| 内心矛盾 | 执着于“做出完美的材料”但反复失败，需要有人告诉她“方向是对的” |

## 2. 外观设计方向

| 维度 | 方向描述 |
| --- | --- |
| 整体印象 | 硬朗实用系，色调以铁灰、深棕、暗橙为主 |
| 发型 | 黑色短发，利落简洁，方便在矿道中活动，有时会沾上矿尘 |
| 服装 | 深棕色厚布工装，外罩皮制护肩和护膝，袖口收紧，方便采掘作业 |
| 配饰 | 腰间挂着矿镐和矿石样本袋，护目镜常挂在脖子上 |
| 随身物品 | 手持矿镐或锤子，背包里装满了矿物样本和简易试验工具 |
| 身高体态 | 中等偏高，身姿挺拔，双手有力，指节上常有矿尘和细小的伤痕 |

## 3. 庇护所设计

| 维度 | 描述 |
| --- | --- |
| 名称 | 炉石工坊 |
| 结构 | 半嵌入山壁的石木混合结构，外墙用原石和木材加固，烟囱常冒着轻烟 |
| 周边 | 矿道入口、矿箱堆、简易冶炼炉、原木支架，地面散落碎石样本 |
| 内部 | 功能至上，熔炉和试验台占主要空间，墙上挂着矿镐和矿物标本 |
| 成长变化 | 矿道入口加固 → 矿箱增多 → 冶炼区扩建 → 试验痕迹和草图出现 → 合金试片和样本陈列 → 增设收纳与展示区 |
| 氛围 | 沉稳、有节奏的敲击声、熔炉火光、金属与矿石的气味 |

## 4. 日常行为模式

| 时段 | 行为 |
| --- | --- |
| 清晨 | 检查矿道安全、整理昨天采掘的矿石 |
| 上午 | 进入矿道采掘，或在地表收集矿石样本 |
| 下午 | 分类整理矿物、进行小型冶炼试验、修补工具 |
| 傍晚 | 在庇护所外堆放和标记新采掘的材料 |
| 夜晚 | 在炉火旁研究矿物样本、绘制试验草图 |
| 雨天 | 在屋内专注进行冶炼或材料测试，矿道内照常工作 |

## 5. 性格深度

梅疏不是一个擅长表达的人。她相信行动比言语更有说服力，一份好的材料比一百句漂亮话更有价值。她的沉默不是冷漠，而是专注——当她面对矿石和熔炉时，那是她最自在的时刻。

她的核心愿望——研制出足够坚硬的合金——源于一种朴素的信念：大地的深处一定藏着更好的东西，只是还没有被找到、被理解。她反复试验、反复失败，但从不放弃。玩家对她来说，最初只是一个可以帮忙搬矿石的人，但随着委托推进，她逐渐发现：这个人不仅帮得上忙，而且理解她在做什么。这对梅疏来说，比任何赞美都重要。

梅疏的亲密表达含蓄而有分量。她不会说“我喜欢你”，但她会把第一块成功的合金试片交给玩家。对她来说，把自己最重要的成果交到一个人手上，就是最大的信任。

## 6. 固定委托详设

### 委托 1：山道口

| 维度 | 描述 |
| --- | --- |
| 阶段 | 熟悉（好感度 100~200） |
| 触发条件 | 玩家好感度达到 100，且当前委托槽为空 |
| 委托目标 | ① 在庇护所矿道入口处放置 8 根火把 ② 用原木加固矿道入口两侧（各放置 4 根原木支架）③ 为梅疏准备基础补给（食物 ×8、木棍 ×16） |
| 任务指引 | 任务卡文字：“矿道口不太安全。如果你手脚不笨的话，帮我把入口弄亮堂些，再加几根撑木。” |

| 主要奖励 | 好感度 +50；解锁「基础矿石」「火把补给」随机委托模板 |
| 协助可能 | 其他玩家可以帮助放置火把和支架 |

## 9. 祝福、跟随与终章方向

| 项目 | 内容 |
