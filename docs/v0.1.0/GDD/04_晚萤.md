# 晚萤（wanying）角色细节设计

## 1. 角色定位

| 属性 | 内容 |
| --- | --- |
| 中文名 | 晚萤 |
| 内部代号 | wanying |
| 生成维度 | 下界（Nether） |
| 维度 | 下界 |
| 核心主题 | 炎热、战斗、近战、防护 |
| 性格特质 | 热烈直率、勇敢无畏、在战斗外有出人意料的温柔面 |
| 说话风格 | 直接利落，有军士般的简洁感，战斗时喊话简短有力，私下来往时偶尔语带暖意 |
| 声音特点 | 声线明亮而有穿透力，节奏快，落字干脆，像火焰爆开的瞬间；战斗时极具压迫感，放松下来后却会露出带一点沙哑的柔和尾音，让人感到她把锋芒收给了信任的人 |
| 核心愿望 | 把危险化为力量，保护自己想保护的人，证明自己不只是为战而活 |
| 内心矛盾 | 习惯了独自战斗所以很难放下警惕，渴望有人并肩但又害怕对方因自己受伤 |

## 2. 外观设计方向

| 维度 | 方向描述 |
| --- | --- |
| 整体印象 | 热烈战斗系，色调以暗红、黑金、橙焰为主 |
| 发型 | 暗红色短发，被下界的炎热气流微微吹起，发梢有不规则的焰色渐变 |
| 服装 | 黑色与暗红色调的轻甲战装，肩部有不对称护甲，左手持盾时护甲更厚重 |
| 配饰 | 腰间挂着烈焰棒制成的短杖，手腕上缠着防火绷带 |
| 随身物品 | 常持剑盾，背包里总有备用武器、防火药水和急救绷带 |
| 身高体态 | 中等身高，深色皮肤，体态精悍有力，站姿随时保持警觉，像一头蓄势待发的猎豹 |

## 3. 庇护所设计

| 维度 | 描述 |
| --- | --- |
| 名称 | 炽灰哨站 |
| 结构 | 用下界砖和黑石建造的坚固哨站式建筑，外墙厚重，有瞭望口 |
| 周边 | 防御栅栏、武器架、旗帜、篝火，地面有战斗训练的痕迹 |
| 内部 | 武备陈列、盾牌与旗帜挂饰、简易锻造台、战利品展示架 |
| 成长变化 | 防御加固 → 武器架增多 → 战利品陈列更丰富 → 增设训练区 → 内部增加更温暖的布置（如织物、灯具）→ 在坚固中透出生活气息 |
| 氛围 | 炽热、坚固、战备感，但内部逐渐有暖意，像一座被守护的堡垒 |

## 4. 日常行为模式

| 时段 | 行为 |
| --- | --- |
| 清晨 | 巡逻庇护所周边、检查防线的完整性 |
| 上午 | 进行战斗训练、维护武器与护甲 |
| 下午 | 主动搜索并清理附近的敌对生物 |
| 傍晚 | 在庇护所门口守望、整理战利品 |
| 夜晚 | 在篝火旁保养武器，偶尔会望着火焰出神 |
| 任何时间 | 当发现威胁时会立刻切换为战斗姿态 |

## 5. 性格深度

晚萤是一个用战斗来定义自己存在价值的人。下界的环境教会了她一件事——不战斗就无法生存。她习惯了一个人面对成群结队的敌人，习惯了在篝火边独自磨刀。但这不意味着她喜欢独处。

她的核心愿望“不只是为战而活”反映了她内心最深处的矛盾：她擅长战斗，但她不想被战斗定义。她希望在战斗之外，还有一个值得回去的地方，有一个值得守护的人。玩家最初在她眼中只是一个“敢来下界的家伙”，但随着委托推进，她开始把玩家从“战友”重新定义为“同行者”。

晚萤的亲密表达热烈而笨拙。她不会说甜言蜜语，但她会把最好的武器留给玩家，会在玩家受伤时第一个冲上来挡住攻击。当她最终接受玩家家园作为“归处”时，她的意思是：我守护过很多地方，但这里是我愿意住下来的地方。

## 6. 固定委托详设

### 委托 1：炽灰落脚

| 维度 | 描述 |
| --- | --- |
| 阶段 | 熟悉（好感度 100~200） |
| 触发条件 | 玩家好感度达到 100，且当前委托槽为空 |
| 委托目标 | ① 在晚萤的庇护所周边放置 8 个防御性方块（栅栏、墙等）② 为她准备基础战斗补给（食物 ×8、治疗药水 ×1）③ 清除庇护所周围的 8 只敌对生物 |
| 任务指引 | 任务卡文字：“看得出来你敢来下界。很好。帮我把附近的防线加固一下，顺便清掉那些乱窜的东西。让我看看你值不值得站在前线。” |
| 完成反馈 | 晚萤检查了防线后简短地说：“还行。至少你不是那种一看到烈焰人转头就跑的类型。” |
| 环境变化 | 庇护所周边增设防御栅栏，威胁被清除 |
| 主要奖励 | 好感度 +50；解锁「基础清剿」「战备补给」随机委托模板 |
| 协助可能 | 其他玩家可以帮助清剿和布置防御 |

| 协助可能 | 其他玩家可陪伴战斗 |
