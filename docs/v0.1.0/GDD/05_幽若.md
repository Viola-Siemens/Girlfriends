# 幽若（youruo）角色细节设计

## 1. 角色定位

| 属性 | 内容 |
| --- | --- |
| 中文名 | 幽若 |
| 内部代号 | youruo |
| 生成维度 | 末地（The End） |
| 维度 | 末地 |
| 核心主题 | 末影、紫颂果、研究、瞬移愿望 |
| 性格特质 | 冷静理性、好奇心旺盛、在研究中忘我、对未知有近乎偏执的热情 |
| 说话风格 | 条理清晰，习惯用“观察”“假设”“结论”的方式表达，但不会显得冷漠 |
| 声音特点 | 声线偏冷静通透，吐字清楚，节奏均匀，像把每个结论都提前在心里验证过一次；谈论实验时会略微加快，提到“回归点”“你身边”时则会放轻声音，显出少见的温柔确定感 |
| 核心愿望 | 提取出能够让人自由瞬移的物质，把末地的诡秘规律变成真正可被理解与应用的成果 |
| 内心矛盾 | 对未知的渴望让她有时忽略了自身安全与生活本身，需要一个“让她记得回来”的坐标 |

| 维度 | 方向描述 |
| --- | --- |
| 整体印象 | 神秘研究系，色调以紫黑、深蓝、银色为主 |
| 发型 | 深紫色长发，松散地披在背后，用一根简单的发绳束住，有时会沾上紫颂果的汁液 |
| 服装 | 深蓝与紫色调的研究袍，外罩一件便于活动的短披肩，袖口有末影珍珠装饰 |
| 配饰 | 脖子上挂着记录本，腰间绑着装满小瓶样本的皮制腰包 |
| 随身物品 | 手持记录本和笔，背包里装满了样本瓶、实验器具和紫颂果 |
| 身高体态 | 中等身高，体态纤细，动作灵巧而准确，眼神中常带着观察和思考的光芒 |

## 3. 庇护所设计

| 维度 | 描述 |
| --- | --- |
| 名称 | 紫影研究所 |
| 结构 | 用紫珀块和末地石建造的研究小屋，外观简洁但有几何设计感 |
| 周边 | 紫颂果种植区、末影珍珠收集装置、简易观测平台 |
| 内部 | 研究台、样本瓶架、记录堆、简易实验设备，墙上贴着观测记录和草图 |
| 成长变化 | 样本瓶增多 → 实验台升级 → 记录与草图覆盖更多墙面 → 实验设备更完善 → 增设“回归坐标”标记装置 → 空间感更强，在理性中透出归属感 |
| 氛围 | 安静、紫色微光、有轻微的空间波动感，像一座漂浮在虚空中的研究所 |

## 4. 日常行为模式

| 时段 | 行为 |
| --- | --- |
| 清晨 | 检查实验样本的状态、记录夜间观测结果 |
| 上午 | 采集紫颂果、观察末影人行为、收集空间波动数据 |
| 下午 | 进行小型实验、整理样本、更新研究记录 |
| 傍晚 | 在观测平台静坐观察末地远景、思考实验方向 |
| 夜晚（末地无昼夜） | 在屋内整理数据、设计新的实验方案 |
| 任何时间 | 发现有趣的末影现象时会立刻开始观察和记录 |

## 5. 性格深度

幽若是一个被好奇心驱动的人。末地对别人来说是一片荒凉的虚空，但对她来说，每一颗紫颂果的瞬移机制、每一个末影人的行为模式、每一道空间的细微波动，都是一个等待被解开的谜题。

她的核心愿望——提取让人自由瞬移的物质——听起来像是疯狂的幻想，但对幽若来说，这是经过无数次观察和实验后得出的严肃假说。她相信末地的规律是可以被理解的，而理解之后就可以被应用。玩家最初在她眼中只是一个“能帮忙带样本的人”，但随着委托推进，她发现玩家不仅提供了研究材料，更重要的是提供了“回来的理由”。当一个人习惯了在虚空中研究未知时，有一个“坐标”能让她记得回到哪里、回到谁身边，这比任何实验数据都珍贵。

幽若的亲密表达是独特的。她不会说“我爱你”，但她会在关键实验成功后说：“我把坐标设在你身边了。”对她来说，把自己的“回归点”设定在一个人身边，就是最郑重的约定。

| --- | --- |
| 阶段 | 熟悉（好感度 100~200） |
| 触发条件 | 玩家好感度达到 100，且当前委托槽为空 |
| 委托目标 | ① 为幽若采集 16 个紫颂果 ② 收集 4 块紫珀块或材料 ③ 将样本放置在庇护所的样本收集区 |
| 任务指引 | 任务卡文字：“你从主世界来的？正好——能帮我收集一些紫颂果和紫珀材料吗？我最近的样本有些不够用了。放在那边的台子上就行，不要碰其他的瓶子。” |
| 完成反馈 | 幽若将紫颂果一个个编号并记录在笔记本上。她抬眼看了玩家一下：“你比我想象中更可靠。至少比末影人可靠——它们从来不肯帮我收集样本。” |
| 环境变化 | 庇护所中样本瓶架上多了新的紫颂果样本 |

| 协助可能 | 其他玩家可以帮助收集样本 |
